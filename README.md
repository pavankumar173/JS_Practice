# JavaScript Practical Examples

This repository contains **JavaScript practical programs using HTML files**.  
Each file demonstrates a specific JavaScript concept for academic and practice purposes.

---

## Aim

To understand and implement basic JavaScript concepts using HTML files.

---

## List of Practicals

1. **Variable_Practical.html**  
   - Demonstrates JavaScript variables and data types

2. **Operators_Practical.html**  
   - Demonstrates arithmetic, comparison, and logical operators

3. **Functions_Practical.html**  
   - Demonstrates function creation and usage

4. **Objects_Practical.html**  
   - Demonstrates JavaScript objects and properties

5. **Arrays_Practical.html**  
   - Demonstrates array creation and operations

6. **Events_Practical.html**  
   - Demonstrates event handling in JavaScript

7. **Animation_Practical.html**  
   - Demonstrates simple animations using JavaScript

---

## Software Requirements

- Any modern web browser (Chrome, Edge, Firefox)
- Text editor (VS Code / Notepad)

---

## Procedure

1. Download or clone the repository.
2. Open any `.html` file in a browser.
3. Observe the output and behavior.
4. Use browser console for debugging.

---

## Result

All JavaScript practical programs execute successfully and demonstrate their respective concepts.

---

## Conclusion

These practical examples help in understanding the fundamentals of JavaScript and its interaction with HTML.
